Reporting Tools

Configuration
-------------
Edit the ``weekly_config.py`` and adapt it to your teams.

Bug Summaries
-------------
Run weekly-bugs.py with the name for the team you want to generate the report about. For instance:
```./weekly-bugs.py ui```
